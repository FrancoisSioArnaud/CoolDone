# CoolDone

CoolDone est une application iOS SwiftUI native, iPhone only, pensée en **widget-first**.

## Objectif V0

Permettre de créer, afficher, modifier et supprimer des Cooldowns récurrents locaux, avec un usage quotidien principalement depuis le widget et une action « C’est fait » via App Intents.

## Stack technique

- iOS natif
- Swift
- SwiftUI
- WidgetKit
- App Intents
- Stockage local à définir dans la V0

## Ouvrir le projet

Ouvrir `CoolDone.xcodeproj` dans Xcode, ou lancer :

```sh
open CoolDone.xcodeproj
```
